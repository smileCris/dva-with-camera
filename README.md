# dva-with-camera
使用Cordova+dva做react移动端开发，并使用相机插件

## 本地开发

```bash
# 本地浏览器开发
$ npm install
$ npm start

# 使用模拟器打开
# 添加 Android 项目
$ cordova platform add android
# Check to see if your system is configured for building android platform.
$ cordova requirements android
# 构建最新的静态文件
$ npm run build
$ cordova run ios android
```
